Overview

A TableView is composed of 3 sub-tables. The table of Column Headers, the table of Row Headers, and the table of table data definitions. Its the responsibility of the TableView to render all Table elements into haml, by providing accessor methods.

The three tables are laid out by TableView as follows: 

  |--------------------------------|
  |                |-------------| |
  |                |             | |
  |                |  column     | |
  |                |  headers    | |
  |                |             | |
  |                |             | |
  |                |-------------| |
  | |-------------||-------------| |
  | |             ||             | |
  | |             ||             | |
  | | row headers || table data  | |
  | |             ||             | |
  | |             ||             | |
  | |-------------||-------------| |
  |           table view           |
  |--------------------------------|
