ValidKeys =

A Hash Array of the supported plist keys for this type. These are plist keys which belong to the PlistType for Info plists. Each CamelCased key name has a corresponding set_or_return method call. For example “CFBundleIdentifier” => c_f_bundle_identifier(value). For more information please see PlistKeyNames
